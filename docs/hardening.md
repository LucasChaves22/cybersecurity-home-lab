# Hardening

## O que é

Hardening é o processo de reduzir vulnerabilidades em sistemas.

## Práticas comuns

- Remover serviços desnecessários  
- Atualizar sistema  
- Configurar firewall  
- Usar senhas fortes  
- Limitar permissões  

## Exemplo prático

- Desativar root no SSH  
- Bloquear portas  
- Configurar firewall  

## Importância

Reduz a superfície de ataque e aumenta a segurança do ambiente.
