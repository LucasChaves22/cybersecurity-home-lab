# Resposta a Incidentes

## O que é

Resposta a incidentes é o processo de identificar, conter e corrigir eventos de segurança.

## Etapas

1. Identificação  
2. Contenção  
3. Erradicação  
4. Recuperação  
5. Lições aprendidas  

## Exemplo prático

- Detecção de acesso suspeito  
- Bloqueio do usuário  
- Análise de logs  
- Correção da falha  

## Importância

Permite reduzir impactos e restaurar o ambiente com segurança.
