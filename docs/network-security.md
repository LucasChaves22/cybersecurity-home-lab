# Segurança de Rede

## Boas práticas

- Alterar senhas padrão de roteadores  
- Utilizar Wi-Fi com criptografia (WPA2/WPA3)  
- Fechar portas desnecessárias  
- Utilizar firewall  
- Monitorar tráfego de rede  
- Segmentar a rede quando possível  
- Atualizar firmware dos dispositivos  

## Controles importantes

- Controle de acesso à rede  
- Monitoramento de tráfego  
- Filtragem de portas  
- Uso de VPN para acesso remoto  

## Exemplos de riscos

- Portas abertas desnecessárias  
- Rede sem criptografia  
- Dispositivos desconhecidos conectados  
- Falta de monitoramento  

## Importância

A segurança de rede é fundamental para proteger a comunicação entre sistemas e evitar acessos indevidos e ataques externos.
