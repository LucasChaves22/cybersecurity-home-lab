# Segurança em Windows

## Boas práticas

- Manter o sistema sempre atualizado  
- Ativar o firewall do Windows  
- Utilizar antivírus ativo  
- Utilizar senhas fortes e complexas  
- Evitar uso de conta administrador no dia a dia  
- Ativar bloqueio automático de tela  
- Desativar serviços desnecessários  

## Controles importantes

- Controle de Conta de Usuário (UAC)  
- Políticas de senha  
- Auditoria de eventos (logs)  
- Permissões de usuários  

## Exemplos de riscos

- Uso de senha fraca  
- Sistema desatualizado  
- Programas desconhecidos instalados  
- Usuário com privilégios excessivos  

## Importância

A segurança em sistemas Windows é essencial para proteger dados, evitar acessos não autorizados e reduzir riscos de ataque.
